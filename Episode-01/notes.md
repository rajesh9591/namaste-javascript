# Episode 1 - How JavaScript Works & Execution Context

## Topics Covered

- JavaScript is single-threaded
- JavaScript is synchronous
- Global Execution Context
- Memory Creation Phase
- Code Execution Phase
- Call Stack

## Key Learnings

- JavaScript creates a Global Execution Context before executing code.
- Memory is allocated first.
- Code is executed line by line.
- Functions get their own execution context.
- The call stack manages execution.
